iOS 112.0.2 版 · 更新於 2017/7/16

隨著 iOS 11 的推出，我們的 iOS 設計套件也更新到了第二版；基於前一版的內容，加入一些蘋果官方設計素材的優點，並額外附帶 Wireframe 常用到的介面元素。

無論你是想簡單地做個草稿來輔助思考、規劃功能、或是進行精細的設計，這款套件都能夠大大地提高你的工作速度。（使用本套件前，請先安裝 SF 字體）

我們是位於台北的 UX / UI / 網頁設計教學組織，這些套件皆是我們的講師親自開發用來教學的素材，並提供給所有人免費使用。如果你對 UI 設計有興趣，可以參考我們所推出的課程，或是造訪我們的部落格與 Facebook 粉絲頁。

套件特點

一百多個經過仔細整理的元件（Symbol），以及覆寫選項（Override）

超過 50 個向量格式的 iOS 系統 icon，每一個皆是精心重繪

元件皆可縮放而不破壞排版，適用於 iPhone SE、7、與 7 Plus 的畫面大小

只需調整幾個關鍵元素，便可改變整個套件的色系，方便探索風格

額外提供繪製 Wireframe 的常用元件

適用於 Sketch 45.2 以上版本 (1MB)
